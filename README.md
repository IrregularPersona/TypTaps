# typtaps

> A minimal Typst editor with live preview.

## Dependencies

- Pdfium binaries.
- The Typst CLI.

## Usage

```bash
cargo run --release
```

## Features

- **Live Watch**: Auto-reloads on file changes (Not super consistent).
- **File Tree**: Navigator for your projects.
- **Integrated Preview**: PDF viewer with zoom and pan (Buggy at best).
